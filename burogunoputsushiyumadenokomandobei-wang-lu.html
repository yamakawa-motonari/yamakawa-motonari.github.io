ブログのプッシュまでのコマンド備忘録

Date 火 31 10月 2017 By 山川去年 Category 備忘録. Tags git github

最低限ブログ投稿に必要なvenvコマンドやgitコマンド

contentやoutputの親ディレクトリで、

source venv/bin/activate

で、(venv) として実行することができる。

同じディレクトリで、

$ ghp-import output
$ git add output/
$ git commit -m 'added all created html files'
$ make html
$ git push https://github.com/username/username.github.io.git gh-pages:master

これで投稿した。 最後のコマンドのあとにgithubでのユーザーネームとパスワードを要求されるので、入力する。
