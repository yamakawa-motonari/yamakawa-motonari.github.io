最低限ブログ投稿に必要なvenvコマンドやgitコマンド

contentやoutputの親ディレクトリで、

source venv/bin/acticate

で、(venv) として実行することができる。

同じディレクトリで、

$ gep-import output
$ git add output/
$ git commit -m 'added all created html files'
$ make html
$ git push https://gitfub.com/username/username.github.io.git gh-pages:master

これで投稿した。 最後のコマンドのあとにgithubでのユーザーネームとパスワードを要求されるので、入力する。

それぞれのgitコマンドの意味については近々調べてメモしておかないと忘れそう…
